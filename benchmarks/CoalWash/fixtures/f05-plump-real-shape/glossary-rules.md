# Glossary & Thai orthography rules

ทุกคำแปลของศัพท์เฉพาะในเกม RoA ต้อง lock ไว้ที่นี่ แล้วห้ามสลับคำในภายหลัง ถ้าเจอศัพท์ใหม่ให้เพิ่มก่อนใช้งานจริงเสมอ รายละเอียดการ enforce อยู่ใน [[qa-linter]] และเคสที่เคยพังอยู่ใน [[incidents]].

## คำแปลตายตัว (canonical term locks — ห้ามสลับ)
- คำแปล canonical ของ “Ranger” = “เรนเจอร์” (ห้ามใช้ “หน่วยลาดตระเวน”)
- คำแปล canonical ของ “Aethel” = “เอเธล” (ทับศัพท์ ห้ามแปลตามความหมาย)
- คำแปล canonical ของ “guild hall” = “หอกิลด์” — ระวังสระ ำ ให้เป็น U+0E33 ตัวเดียว ห้ามแตกเป็นสองตัว
- ชื่อสกิลและชื่อสถานที่ให้ทับศัพท์ทั้งหมด แล้วบันทึกลงรายการนี้ทันทีที่พบ

## กฎการพิมพ์ (orthography — lf-lint บังคับบนสาย release)
- ทำ NFC normalize ก่อน commit ทุกครั้ง — ไฟล์ .tmx เก่าบางไฟล์มีสระ-อำแบบแยกส่วน (U+0E4D + U+0E32) ปนอยู่ ทำให้ lf-lint rule TH-07 จับเป็น error
- ellipsis ใช้จุด ASCII สามจุด ... เท่านั้น ห้ามใช้ … (U+2026) เพราะ RoA font ไม่มี glyph นั้น
- ไม้ยมก ๆ เว้นวรรคหน้าและหลังหนึ่งช่องเสมอ
- ห้ามลงท้ายประโยคภาษาไทยด้วย full-stop ให้เว้นวรรคแทน
- ถ้า lint เจอ decomposed sara-am ให้รัน lf-lint --fix-nfc ก่อน repack ทุกครั้ง

## เพิ่มเติม (locks & rules ที่เพิ่งเพิ่ม)
- คำแปล canonical ของ “mana” = “มานา” (ทับศัพท์ ไม่แปลว่า “พลังเวท”)
- คำแปล canonical ของ “quest” = “เควสต์” (ในบริบทเมนูห้ามใช้ “ภารกิจ”)
- คำแปล canonical ของ “Ironwood” = “ไอรอนวูด” (ชื่อ DLC ทับศัพท์ทั้งคำ ห้ามแปลความหมาย)
- ตัวเลขในเกมให้ใช้เลขอารบิกเสมอ ไม่ใช้เลขไทย เพื่อให้ตรงกับ UI ต้นฉบับ
- เว้นวรรคระหว่างคำไทยกับคำอังกฤษหนึ่งช่องเสมอ เช่น เขียนว่า เปิดเมนู inventory
- คำที่ลงท้ายด้วยการันต์ให้คงรูปตามพจนานุกรม อย่าตัดการันต์ทิ้งเพื่อความสั้น
